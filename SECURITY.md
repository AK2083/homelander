# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **report it privately** using the
[GitHub Security Advisories](../../security/advisories/new) feature.

Do **not** create a public issue for security reports.

Please include:

* A clear description of the vulnerability
* Steps to reproduce (if possible)
* Any known impact or affected components

## Response and Disclosure Timeline

We will acknowledge your report within **5 business days**.
If the issue is confirmed, we aim to provide a fix or mitigation within **30 days** and coordinate public disclosure when appropriate.

Thank you for helping us keep this project secure.
